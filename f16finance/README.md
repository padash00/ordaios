# Orda Point — Система управления точками продаж

**Orda Point** — комплексная система управления бизнесом для многоточечных розничных и сервисных операций. Разработана для казахстанского рынка.

Состоит из двух частей:
- **Веб-портал** (Next.js) — административная панель для владельцев и менеджеров
- **Десктопное приложение** (Electron) — терминал для кассиров и операторов

## Возможности системы

- **Многоточечность** — управление несколькими локациями из одного интерфейса
- **Офлайн-работа** — терминал работает без интернета, синхронизируется при подключении
- **Реальное время** — данные из терминалов отображаются мгновенно в веб-панели
- **Telegram-интеграция** — сменные отчёты и уведомления через бот
- **Контроль доступа** — роли: владелец, менеджер, оператор, администратор

## Десктопное приложение (Electron)

Терминал устанавливается на кассовый компьютер в точке продаж.

### Страницы приложения

| Страница | Описание |
|---|---|
| **Вход** | Авторизация оператора по коду или паролю |
| **Выбор точки** | Выбор рабочей локации (если оператор работает в нескольких точках) |
| **Смена** | Главный экран: ввод выручки (наличные, карта, Kaspi), расчёт ФАКТ, отправка сменного отчёта |
| **Магазин / Продажи** | POS-терминал: поиск товара по штрихкоду или названию, корзина, оплата |
| **Возвраты** | Оформление возвратов покупателю |
| **Заявки** | Запрос товаров со склада |
| **Мой кабинет** | Профиль оператора: история смен, зарплата, задачи |
| **Настройка** | Первичная настройка устройства (URL сервера, токен) |

### Режимы работы терминала

Каждая точка может работать в одном из режимов:
- **Сменный отчёт** — только ввод выручки и отправка отчёта
- **Магазин** — полноценный POS с товарами, корзиной, чеками

Режим настраивается в веб-панели в разделе **Устройства**.

### Автообновление

Приложение проверяет обновления при каждом запуске. Если выходит новая версия — скачивается в фоне и устанавливается при следующем закрытии программы.

## Разработка

```bash
# Установка зависимостей (веб)
npm install

# Запуск веб-портала
npm run dev

# Запуск Electron-приложения
cd desktop/electron_point
npm install
npm run dev
```

### Выпуск новой версии Electron-приложения

```bash
cd desktop/electron_point

# Обновить версию (patch / minor / major)
npm version patch

# Запушить тег — GitHub Actions автоматически соберёт и опубликует .exe
git push origin --tags
```
